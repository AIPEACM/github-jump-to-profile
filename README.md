# GitHub Home → Profile

A simple Tampermonkey script that redirects the GitHub logo to your profile page instead of the home feed.

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/)
2. Download from [Greasefork](https://greasyfork.org/en/scripts/573226-github-home-profile)
3. Click **Install this script** in Tampermonkey

## What it does

By default, clicking the GitHub logo takes you to the home feed (`/`). This script redirects it to your profile page (`/@YOUR_USERNAME`) instead.

- Automatically detects your username from GitHub's page data
- Works across all GitHub navigations
- No configuration needed
